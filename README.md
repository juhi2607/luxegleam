 💎 LuxeGleam - Luxury Jewelry E-commerce Website

LuxeGleam is a modern luxury jewelry e-commerce website built using **React.js** and **Material UI (MUI)**.  
It includes cart management, wishlist, checkout flow, coupon system, Razorpay test integration, and PDF invoice generation.

---
🚀 Live Demo

Deployed on Vercel  
🔗 https://luxegleam.vercel.app

---
 🛠 Tech Stack

- React.js
- Material UI (MUI)
- React Router DOM
- Context API (Cart & Wishlist)
- Razorpay (Test Mode)
- jsPDF (Invoice generation)
- Vercel (Deployment)
- GitHub (Version Control)

---
 ✨ Features
 🛍 Shopping Features
- Product listing page
- Product detail page
- Category filtering
- Search functionality
- Price sorting
- Wishlist functionality
- Add to cart
- Quantity increase/decrease
- Remove item from cart
  
 🛒 Cart System
- Persistent cart using localStorage
- Subtotal calculation
- Grand total calculation
- Real-time updates

 💳 Checkout System
- Coupon code support (LUXE10 – 10% discount)
- Razorpay test payment integration
- Order summary page
- Order success page
